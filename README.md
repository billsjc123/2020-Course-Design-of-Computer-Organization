## **硬连线控制器设计**

按照给定数据格式、指令系统和数据通路，根据所提供的器件要求，自行设计一个基于硬布线控制器的顺序模型处理机。

- 实现一：根据设计方案在TEC-8上进行组装、调试、运行
- 实现二：仿真模拟向量测试
- 附加功能：
  - 在原指令基础上要求扩指至少三条
  - 修改PC指针功能（任意指针）
